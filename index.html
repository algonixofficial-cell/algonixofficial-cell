1. HTML কী? (Short)

HTML = HyperText Markup Language. Web page-er structure define kore — content gula ke tags diye mark up kore. CSS style dey, JavaScript interaction dey.

Exercise 1: নীচের সিম্পল HTML ফাইলটি কপি করে আপনার কম্পিউটারে index.html নামে সেভ করে ব্রাউজারে খুলুন। 
<!doctype html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <title>My first page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>Ami HTML shikhchi.</p>
  </body>
</html>

2. Basic structure

Pratyek HTML dokument-er main parts: <!doctype>, <html>, <head>, <body>.

  • <head> — meta, title, link to CSS, fonts.
  • <body> — visible content (headings, paragraphs, images...).
Exercise 2: title, meta name="description" যোগ করুন।

3. Text, headings, paragraphs & lists

Headings: <h1> to <h6>. Paragraphs: <p>. Lists: ordered <ol> এবং unordered <ul> এবং list item <li>.

<h1>Main title</h1>
<p>This is a paragraph.</p>
<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

4. Images & media

Image tag: <img src="..." alt="..."> — alt important for accessibility. Video and audio tags exist too.

<img src="sunflower.jpg" alt="Sunflower" width="400">

<video controls>
  <source src="video.mp4" type="video/mp4">
  Your browser doesn't support video tag.
</video>

5. Links & navigation

Anchor tag: <a href="URL">Link text</a>. Internal link uses fragment (#id).

<a href="https://example.com" target="_blank" rel="noopener">Visit example</a>
<a href="#projects">Jump to projects</a>

6. Forms — user input

Form tag with inputs, labels, select, textarea, button. Always pair <label for="id"> with inputs for accessibility.

<form action="/submit" method="post">
  <label for="name">Name</label>
  <input id="name" name="name" type="text" required>
  <button type="submit">Send</button>
</form>
Exercise 3: Create a contact form with name, email, message and a submit button. Try pressing submit (it will attempt to post — that's fine).

7. Semantic HTML

Use proper tags for better structure and accessibility: <header>, <nav>, <main>, <article>, <section>, <aside>, <footer>.

8. Intro to CSS (quick)

CSS adds style. Add with <style> in head or external <link rel="stylesheet" href="style.css">.

<style>
  body{font-family:Arial}
  h1{color:darkblue}
</style>

Practice: change font-size, colors, spacing.

9. Intro to JavaScript (quick)

JS makes pages interactive. Add with <script>. Example below shows a click handler.

<button id="btn">Click me</button>
<script>
  document.getElementById('btn').addEventListener('click', function(){
    alert('Hello!')
  })
</script>
